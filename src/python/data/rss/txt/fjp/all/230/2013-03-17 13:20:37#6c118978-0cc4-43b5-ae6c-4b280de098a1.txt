تنفيذ 15 ألف حكم قضائي وضبط 85 قطعة سلاح خلال 48 ساعة
 
نجح قطاع مصلحة الأمن العام على مستوى الجمهورية وبالتنسيق مع مديريات الأمن وقوات الأمن المركزى من ضبط 85 قطعة سلاح نارى، وتنفيذ 15081 حكم قضائي متنوع، وإعادة 33 سيارة مبلغ بسرقتها خلال 48 ساعة فى إطار مواصلة الحملات الأمنية المكثفة لمواجهة أعمال البلطجة وضبط الخارجين عن القانون وحائزى الأسلحة النارية والبيضاء وإحكام السيطرة الأمنية.
وتم ضبط (85) قطعة سلاح نارى عبارة عن (5) بنادق آلية، و(2) رشاش، و(25) بندقية غير مششخنة، و( 1 ) بندقية مششخنة، و( 7 ) طبنجات، و( 46 ) فرد محلى الصنع ، و(478) طلقة نارية مختلفة الأعيرة، بالإضافة إلى ضبط (331 ) قطعة سلاح أبيض.
وفى قضايا المخدرات تم ضبط ( 220 ) متهم فى قضايا مخدرات ضبط خلالها 77,445 كيلو جرام بانجو، و( 48 ) شجرة من نبات البانجو المخدر، و8,882 كيلو جرام حشيش، و185 جرام هيروين، و139 جرام أفيون، و( 12163 ) قرص مخدر.
أما التشكيلات العصابية فتم ضبط  ( 4 ) تشكيلات ضمت ( 18 ) متهما ارتكبوا ( 11 ) حادث سرقة متنوعة، وضبط ( 8 ) متهمين لقيامهم بإرتكاب أعمال البلطجة وحوادث السرقات بالإكراه وبحوزتهم ( 1 ) طبنجة، و( 6 ) قطع سلاح أبيض، بالإضافة إلى ضبط ( 4 ) سجناء هاربين.
وفى مجال تنفيذ الأحكام: تم تنفيذ ( 15081 ) حكمًا قضائيا متنوعا هم ( 68 ) حكم جناية، و(6493) حكم حبس جزئى، و(1299) حكم حبس مستأنف، و(4552) حكم غرامة، و(2669) حكم مخالفة ، و(87) دراجة بخارية مخالفة.
وفى مجال ضبط السيارات المُبلغ بسرقتها تم ضبط ( 33 ) سيارة مُبلغ بسرقتها.
