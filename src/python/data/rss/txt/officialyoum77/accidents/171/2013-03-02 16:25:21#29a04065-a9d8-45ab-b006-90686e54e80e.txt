ضبط 41 محكوما عليهم وتحرير 364 مخالفة مرورية فى حملة بالإسماعيلية
تمكنت حملة أمنية من ضبط 41 محكوما عليهم فى قضايا متنوعة وتحرير 364 مخالفة مرورية وإشغال طريق تم التحفظ على المتهمين وتحرير محاضر بالمخالفات، وجارٍ العرض على النيابة.
وكان اللواء محمد عيد، مدير أمن الإسماعيلية، قد تلقى إخطارًا من اللواء محمد العنانى نائب مدير الأمن يفيد بتمكن مباحث تنفيذ الأحكام بمديرية أمن الإسماعيلية من ضبط 41 محكوما عليهم فى 164 قضية متنوعة، كما تمكنت شرطة المرافق من رفع عدد ( 25 ) حالة إشغال طريق وإعدام عدد ( 10 ) كلاب ضالة.
كما تم توجيه حملات مرورية مكثفة ضمت ضباط إدارتى المرور والبحث الجنائى استهدفت المحاور الرئيسية بالمدينة والأسواق والمناطق ذات الكثافة السكانية لضبط كافة المخالفات خاصةً الدراجات النارية غير المُرخصة، حيث تم ضبط عدد ( 170 ) مخالفة مرورية متنوعة ( نموذج 125 ) وسحب عدد ( 44 ) رخصة ما بين قيادة وتسيير وضبط عـدد ( 35 ) مخالفة سرعة ( رادار ) وضبط عدد ( 17 ) مخالفة انتظار خاطئ وضبط عـدد ( 15 ) مخالفة شروط الترخيص" تحميل ركاب بالأجر"وحجز عدد ( 1 ) دراجة بدون ترخيص وحجز عدد ( 1 ) سيارة بدون ترخيص وإزالة عدد ( 40 ) حالة لملصقات مخالفة وضبط عدد ( 3 ) مخالفات عدم كتابة بيانات وضبط عدد ( 3 ) مخالفات بيئة ( دخان ).
