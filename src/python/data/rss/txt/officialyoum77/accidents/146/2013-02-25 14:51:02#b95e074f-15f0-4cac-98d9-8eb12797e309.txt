ضبط 48 هاربا من أحكام وتحرير 345 مخالفة مرورية وآداب بالإسماعيلية
تمكنت حملة أمنية مكبرة ضمت مباحث تنفيذ الأحكام والمباحث الجنائية وشرطة المرافق ومباحث الآداب ومباحث المرور من ضبط 48 هاربا من 180 حكما متنوعا وتحرير 345 مخالفة مرورية وإشغال طريق وآداب، وتم التحفظ على المتهمين، وتحرير محاضر بالمخالفات والعرض على النيابة.
وكان اللواء محمد عيد مدير أمن الإسماعيلية قد تلقى إخطارا من اللواء محمد العنانى نائب مدير الأمن يفيد بتمكن مباحث تنفيذ الأحكام بالإسماعيلية من ضبط 48 هاربا من 180 حكما متنوعا، وقامت مباحث الآداب بضبط عدد ( 2 ) محل بدون ترخيص وضبط عدد ( 2 ) بائع متجول وضبط عدد ( 2 ) تدخين فى الأماكن العامة، كما تمكنت شرطة المرافق من رفع عدد ( 20 ) حالة إشغال طريق.
وقامت مباحث المرور بضبط عدد ( 151 ) مخالفة مرورية متنوعة ( نموذج 125 ) وضبط عـدد ( 45 ) مخالفة سرعة ( رادار ) وسحب عدد ( 36 ) رخصة ما بين قيادة وتسيير وإزالة عدد ( 35 ) حالة لملصقات مخالفة وضبط عدد ( 28 ) مخالفة انتظار خاطئ وضبط عـدد ( 11 ) مخالفة شروط الترخيص " تحميل ركاب بالأجر " وحجز عدد ( 7 ) دراجة بدون ترخيص وحجز عدد ( 3 ) سيارة بدون ترخيص وضبط عدد ( 3 ) مخالفة عدم كتابة بيانات.
