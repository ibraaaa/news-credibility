مرور الإسماعيلية يضبط 356 مخالفة مرورية فى حملات مكثفة
تمكنت حملة مرورية بالإسماعيلية من ضبط 356 مخالفة مرورية، وتم تحرير محاضر بالمخالفات، وجارى العرض على النيابة.
وكان اللواء محمد عيد مدير أمن الإسماعيلية قد تلقى إخطارا من المباحث يفيد بأنه تم القيام بحملات مرورية مكثفة ضمت ضباط إدارتى المرور والبحث الجنائى استهدفت المحاور الرئيسية بالمدينة والأسواق والمناطق ذات الكثافة السكانية لضبط كافة المخالفات خاصةً الدراجات النارية الغير مُرخصة، أسفرت عن ضبط عدد ( 166 ) مخالفة مرورية متنوعة ( نموذج 125وسحب عدد ( 51 ) رخصة ما بين قيادة وتسيير وضبط عـدد ( 55 ) مخالفة سرعة ( رادار) وإزالة عدد ( 39 ) حالة لملصقات مخالفة وضبط عدد ( 22 ) مخالفة انتظار خاطئ وضبط عـدد ( 8 ) مخالفة شروط الترخيص " تحميل ركاب بالأجر وحجز عدد ( 2 ) دراجة بدون ترخيص وحجز عدد ( 2 ) سيارة بدون ترخيص وضبط عدد ( 1 ) مخالفة بيئة "دخان".
